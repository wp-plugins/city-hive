City Hive partner program Wordpress widget

== Description ==

City Hive platform augments any website with a fully functional, universal  point of sale and delivers targeted offers that can be sold directly to end users without redirection to a 3rd party site.
The City Hive platform utilizes its patent pending technology along with location sensitivity, proprietary machine learning algorithms and data analytics in order to leverage the power of the companyâ€™s rapidly expanding retail network and enable the sale of wine and spirits online.
City Hive's turnkey solution enables clients to utilize existing infrastructure to monetize content and/or sell products online.

The [City Hive](http://cityhive.net/wordpress "City Hive WP page") affiliate program plugin allows partners to automatically present content related product on their blog posts and earn money promoting local products to their readers.
The WordPress plugin automatically presents related product according to each blog post content and visitor's location.
City Hive plugin allows to manually search and set related product through the edit post administration panel. City Hive plugin match any device and any design.

Current locations:

* New York City


== Installation ==

1. Upload `city-hive` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
